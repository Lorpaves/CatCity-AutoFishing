# CatCity-AutoFishing

猫之城自动钓鱼

## 下载python

[python.org](https://python.org)

### 下载jupyter notebook

```shell
pip install jupyter
```

### 下载模块

```shell
pip install opencv-python

pip install pillow

pip install pywin32

pip install numpy

pip install ddddocr
```

### 启动Jupyter Notebook

```shell
jupyter notebook
```
