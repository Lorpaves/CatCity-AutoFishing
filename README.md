# CatCity-AutoFishing

猫之城自动钓鱼

## 下载

[python.org](https://python.org)

[雷电模拟器9 64位](https://www.ldmnq.com/)

### 下载jupyter notebook

```shell
pip install jupyter
```

### 下载模块

```shell
pip install opencv-python pillow pywin32 numpy ddddocr
```

### 启动Jupyter Notebook

```shell
jupyter notebook
```

### 调整分辨率

模拟器分辨率：1280x720

桌面分辨率：1920x1080

修改按键设置，向下滑动为space键
